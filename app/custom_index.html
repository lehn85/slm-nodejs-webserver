A solar panel monitoring station in Minsk, Belarus

This is a test project. Solar panel is very small (60mmx30mm). Temperature and humidity (outdoor) are measured along with voltage and output power per m2. Esp8266 is used to measure data from sensors. Data is sent to Node-red server running in a local server before being forwarded to server on internet. Time is presented in local time in Belarus (GMT+3).
